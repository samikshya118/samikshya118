## things i've built (so far)

**🎙️ CORA — conversational voice agent** *(Aug 2025)*
real-time voice AI: AssemblyAI STT → Google Gemini → Murf TTS, all over WebSocket.
session memory + live web search + FastAPI backend. voice-to-voice, zero typing needed.

**🔍 fraud detection system** *(Sep 2025)*
1.75M+ financial records. extreme class imbalance handled via threshold optimization.
99% accuracy · 89% recall · 0.87 PR-AUC. built to catch what others miss.

**🐾 animal image classification** *(Sep 2025)*
CNN-based classifier across 15 animal categories.
data augmentation + precision/recall/F1 + confusion matrix eval. clean, reproducible pipeline.

**🏡 airbnb data analysis** *(Jan 2026)*
deep EDA uncovering pricing patterns, location trends, and booking behaviour.
the kind of analysis that actually tells a story.

**🛒 customer shopping behaviour analysis** *(Dec 2025)*
retail behaviour patterns decoded through feature analysis and visual storytelling.
